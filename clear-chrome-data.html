🔧 Chrome 브라우저 Dailyfeed 데이터 초기화

⚠️ Chrome에서 문제가 발생하는 이유:

  • 이전 JWT 토큰이 localStorage에 남아있음
  • 오래된 쿠키(refresh_token)가 남아있음
  • 캐시된 데이터가 충돌을 일으킴

1. 현재 저장된 데이터 확인

2. LocalStorage 초기화

3. 쿠키 확인 및 삭제

4. 전체 초기화

5. Chrome 개발자 도구로 직접 초기화

Chrome에서 직접 초기화하는 방법:

  1. F12 또는 우클릭 → 검사 클릭
  2. Application 탭 선택
  3. Storage → Clear site data 클릭
  4. 또는 Console에 아래 명령어 입력:
// LocalStorage 초기화
localStorage.clear();

// SessionStorage 초기화
sessionStorage.clear();

// 특정 쿠키 삭제
document.cookie = "refresh_token=; expires=Thu, 01 Jan 1970 00:00:00 UTC; path=/;";

// 페이지 새로고침
location.reload();
